# &lt;PassOp/> - Your own Password Manager

## Description

PassOp is a simple and responsive password manager app built with React. This app enables users to securely store, manage, and retrieve their website or app credentials. It supports functionality to add, edit, and delete entries, with data stored in the browser's local storage.

## Features

- **Add New Entries**: Save new website or app names, usernames, and passwords.
- **Edit Existing Entries**: Modify existing credentials.
- **Delete Entries**: Remove credentials that are no longer needed.
- **Responsive Design**: Fully functional on both desktop and mobile devices.
- **Local Storage**: Stores data securely in the browser's local storage.

## Installation

To get started with PassOp locally, follow these steps:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/akashjitu18/PassOp-password-manager.git

2. **Navigate to the Project Directory**

   ```bash
   cd PassOp-password-manager

3. **Install Dependencies**

   ```bash
   npm install

4. **Start the Development Server**

   ```bash
   npm run dev
The app will be available at http://localhost:5173.

## Usage

1. **Add an Entry**

   - Enter the website or app name, username, and password.
   - Click "Save" to store the entry.

2. **Edit an Entry**

   - Click on the "Edit" button next to the entry you want to modify.
   - Update the details and click "Save".

3. **Delete an Entry**

   - Click on the "Delete" button next to the entry you want to remove.
